# Agent Log 

I used Antigravity, Claude, Gemini and Chatgpt AI tools while doing this project.
I explained everything what I want, it generated code and suggestions and after that I modifed the code as i want, and tested it whether it is working without errors or not.

## How I Used AI tools

  **Project setup**
  I used it to set up the frontend , backend , MongoDB connection . This saved my time .

 **Frontend components**
  I used it to generate basic UI components like dashboard pages , forms, and the landing page layout. After that adjusted styling and structure based on what I needed.

 **Debugging**
  I used AI when errors occured, to identify reasons why those are occured . For example, I observed a bug where jobs were only seeded for one company. AI helped me to identify the issue, later I verified and fixed those issues myself.

 **Styling and design**
  I used AI to experiment with Tailwind styles (gradients, glassmorphism, etc...) and then refined the UI manually.

 **Testing in the browser** 
 It has a built-in browser tool, so I used it to visually verify that pages rendered correctly or not, buttons worked or not, search/filter functioned or not, and the layout is mobile responsive or not, etc..

## Refinements and Fixes

 These are some things that fixed and refined :

* Seed logic initially only worked for one company 
* Enum validation errors in job schema
* Accessibility issue (buttons were not keyboard accessible)
* CORS setup between frontend and backend

## What I Learned
  These are the things that I learned while doing this project:

* I learned that AI is very useful for speed, especially for repetitive code
* I learned that it is easy to accept wrong output if you don’t review properly. So we have to make sure it is not halucinating.
* I learned that understanding code is still important, especially for debugging.
* I learned that small things like accessibility and schema design matter more than expected.

## Final Thoughts

AI helped me to build this project faster, but all major decisions like architecture, schema design, and UX were taken by me. And In some cases, I rewrote or adjusted AI output to better fit the project.
